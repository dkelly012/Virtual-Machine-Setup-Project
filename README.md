# Virtual Machine Setup Lab

## Overview
This project demonstrates how I set up and configured virtual machines for cybersecurity testing and lab environments.

## Objective
The goal was to create a working lab environment using virtualization software to run multiple operating systems safely.

## Tools and Technologies
- VMware / VirtualBox
- Windows 11 Virtual Machine
- Kali Linux Virtual Machine

## Process
- Installed virtualization software
- Created a Windows 11 VM
- Installed Kali Linux VM
- Configured network settings between machines
- Tested connectivity between VMs

## Key Skills Demonstrated
- Virtual machine setup and configuration
- Operating system installation
- Network configuration
- Troubleshooting system errors


## What I Learned
I learned how to build a safe testing environment using virtual machines. This is important in cybersecurity because it allows testing tools and attacks without affecting real systems.
